# 📋 TaskBot — Телеграм-бот для управления задачами

## Что умеет
- Менеджер добавляет задачи с дедлайном конкретному сотруднику
- Команда получает список задач каждое утро в 10:00
- За час до дедлайна — автоматическое напоминание
- Сотрудник нажимает «✅ Готово» — менеджер сразу видит уведомление

---

## Шаг 1 — Создать бота

1. Открой Telegram, напиши @BotFather
2. Отправь `/newbot`
3. Придумай имя (например: `Мои Задачи`) и username (например: `mytasks_manager_bot`)
4. Скопируй **токен** — он выглядит так: `7123456789:AAF...`

---

## Шаг 2 — Узнать свой Telegram ID

Напиши боту @userinfobot — он пришлёт твой числовой ID (например: `123456789`)

---

## Шаг 3 — Задеплоить на Railway (бесплатно)

1. Зарегистрируйся на [railway.app](https://railway.app)
2. Нажми **New Project → Deploy from GitHub**
3. Загрузи папку с файлами бота в GitHub (или используй Railway CLI)
4. В разделе **Variables** добавь:

```
BOT_TOKEN=твой_токен
MANAGER_ID=твой_telegram_id
TIMEZONE=Europe/Moscow
MORNING_TIME=10:00
```

5. Railway сам запустит бота

---

## Альтернатива — запуск локально

```bash
# Установить зависимости
pip install -r requirements.txt

# Создать файл .env (скопируй из .env.example и заполни)
cp .env.example .env

# Запустить
python bot.py
```

## Команды бота

### Для менеджера:
| Команда | Что делает |
|---------|-----------|
| `/task` | Добавить задачу |
| `/tasks` | Все активные задачи |
| `/done_list` | Выполненные задачи |
| `/team` | Список команды |

### Для сотрудников:
| Команда | Что делает |
|---------|-----------|
| `/start` | Зарегистрироваться |
| `/mytasks` | Мои задачи |

---

## Как пользоваться

**Добавить задачу (менеджер):**
1. `/task` → выбрать сотрудника → написать название → написать дедлайн `25.06 18:00`
2. Сотрудник сразу получает уведомление

**Отметить выполнение (сотрудник):**
- Нажать кнопку «✅ Готово» под задачей
- Менеджер получает уведомление
